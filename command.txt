**Common Command Lines**

* pwd (print working directory)
  displays the file path or your current directory location

* cd [loc] (change directory)
  allows you to move throughout the filesystem of a computer

* mkdir (make directory)
  creates new folder in the current directory

* ls
  lists the contents of the current directory

* touch [filename]
  creates a new file in the current working directory with the name provided

* rm [filename]
  removes a file

---------------------

Repository -- like a folder/container that stores files/folders related to a project. It keeps track of the changes made over time.

  > Local Repository -- version of the repo that is store on your own computer
  > Remote Repository -- version of the repo that is store on a server or cloud-based platform [Github]

* git init 
	- initialize a folder

* git config --global user.name "<username>"
* git config --global user.email "<email address>"
	- sets the user's username and email address

*git config --global user.name
*git config --global user.email
	- to check if username and email address were successfully connected

*git status 
	- displays the current status of your file/folder

*git add .
	- adds all modified files in your current directory to the staging area

* git add "filename"
	- adds a specific file to the staging area

* git commit -m "message"
	- saves the changes that you've made to the files in the staging area to the repo. The "message" describes what changes were made in this commit.

* git log 
	- displays all the commits that were made

* git log --oneline
	- condense the response of git log

* git remote add origin [paste the https link from github]

* git push -u origin master

* git push 
	- uploading files/folder from local repo to remote repo

* git clone [paste the https link from github]
	- creating a copy of a remote repo on your local repo

* git pull 
	- retrieving changes from a remote repo and merges them into your local repo

* git fork
	- creating a copy of an existing repo on a remote hosting platform [GitHub]
